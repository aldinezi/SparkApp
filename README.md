# spark-app

This app was made as an Angular-Electron final Spark school project in June 2017.

## To Use

```bash
# Clone this repository
git clone https://github.com/aldinezi/SparkApp.git
# Go into the repository
cd SparkApp
# Install dependencies
npm install
# Run the app
npm start
```
